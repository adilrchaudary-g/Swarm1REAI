## 2. PREMIUM UI — LINEAR / VERCEL / STRIPE AESTHETIC

### Mantlr Analysis
- ![Premium UI Principles](https://cdn.sanity.io/images/q4asopm6/production/be5fd729e9027d79d50626149f4a2d85c269a79e-1080x649.jpg)

**Six Critical Decisions from Linear/Vercel/Stripe:**
1. Interaction Density — every component has complete microstates
2. Typography as Brand Anchor — one typeface family used systematically
3. Color Restraint — color for meaning, not decoration
4. Crafted Microstates — hover, focus, active, disabled, loading all designed
5. Physical Metaphor Respect — coherent spatial logic in animations
6. Obsession with Specific Cases — context-specific feel

Source: [How Stripe, Linear, and Vercel Ship Premium UI](https://mantlr.com/blog/stripe-linear-vercel-premium-ui)
